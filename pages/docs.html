Getting Started

Welcome to Aung Myo Kyaw Developer Platforms! This guide will help you get started with our platform and make the most of your development experience.

Create an Account

To access all features, you need to create an account. Click the Register button in the side menu and fill in your details.

1. Click on Register in the side menu
2. Enter your email and password
3. Verify your email address
4. Login to access all features

Explore Platforms

Browse through various development platforms tailored for different technologies and skill levels.

Tip: Start with the Web Development Hub if you're new to programming!

Join Projects

Contribute to open-source projects or start your own. Collaborate with developers worldwide.

API Reference

Access our platform's API to build custom integrations.

Authentication

POST /api/auth/login
Content-Type: application/json

{
    "email": "user@example.com",
    "password": "yourpassword"
}

Get Platforms

GET /api/platforms
Authorization: Bearer YOUR_TOKEN

Frequently Asked Questions

How do I reset my password?

Click on "Forgot Password" on the login page and enter your email address. You'll receive a password reset link.

Can I create my own platform?

Yes! Contact admin through the Connect page to discuss adding your platform to our collection.

Is there a mobile app?

We're currently developing mobile apps for iOS and Android. Stay tuned for updates!

How can I contribute to projects?

Each project has a GitHub repository. Fork the repository, make your changes, and submit a pull request.
